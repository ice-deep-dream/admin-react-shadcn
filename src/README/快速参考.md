# 快速参考卡片

## 常用命令

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm run dev

# 生产构建
pnpm run build

# 预览构建
pnpm run preview

# 代码检查
pnpm run lint

# 代码格式化
pnpm run format
```

## Git 提交规范

```
<type>(<scope>): <subject>
```

**类型**:
- `feat` - 新功能
- `fix` - Bug 修复
- `refactor` - 代码重构
- `docs` - 文档更新
- `chore` - 构建/工具
- `test` - 测试
- `perf` - 性能优化

**示例**:
```
feat(auth): add email login
fix(users): handle null user
refactor: extract service logic
docs: update API guide
```

## 项目结构

```
src/
├── components/      # UI 组件
├── features/        # 功能模块
├── context/         # 状态管理
├── hooks/           # 自定义 Hook
├── lib/             # 工具库
├── locales/         # 国际化

└── README/          # 文档
```

## 国际化

**默认语言**: 中文 (zh)

**翻译文件**: `src/locales/`

**使用方式**:
```typescript
import { useTranslation } from 'react-i18next'

const { t } = useTranslation()
t('common.dashboard')
```

## 主题

**主题提供者**: `theme-provider.tsx`

**主题类型**: 深色、浅色、系统

**使用方式**:
```typescript
import { useTheme } from '@/context/theme-provider'

const { theme, setTheme } = useTheme()
```

## 路由

**路由库**: TanStack Router

**路由文件**: `src/routes/`

**嵌套路由**: 使用 `(group)` 语法

## 表格系统

**组件位置**: `src/components/data-table/`

**主要功能**:
- 排序
- 过滤
- 分页
- 批量操作
- 行操作

## 国际化翻译键命名

```
common.xxx          # 通用翻译
```

## 组件命名

- **文件名**: PascalCase
- **组件名**: 与文件名一致
- **Props**: 使用接口定义

## 样式规范

- 使用 Tailwind CSS
- 避免内联样式
- 保持样式一致性
