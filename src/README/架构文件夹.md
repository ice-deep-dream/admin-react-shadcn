# 架构文件夹

## 项目结构

```
shadcn-admin/
├── public/                 # 静态资源
│   └── images/            # 图片资源
├── src/
│   ├── assets/            # 资源文件
│   │   ├── brand-icons/   # 品牌图标
│   │   └── custom/        # 自定义图标
│   ├── components/        # UI 组件
│   │   ├── data-table/    # 数据表格组件
│   │   └── layout/        # 布局组件
│   │       ├── data/      # 布局数据
│   │       └── *.tsx      # 布局组件
│   ├── config/            # 配置文件
│   ├── context/           # React Context
│   ├── features/          # 功能模块
│   │   ├── apps/          # 应用管理
│   │   ├── auth/          # 认证模块
│   │   ├── chats/         # 聊天模块
│   │   ├── dashboard/     # 仪表盘
│   │   ├── errors/        # 错误页面
│   │   ├── settings/      # 设置模块
│   │   ├── tasks/         # 任务管理
│   │   └── users/         # 用户管理
│   ├── hooks/             # 自定义 Hook
│   ├── lib/               # 工具库
│   ├── locales/           # 国际化翻译文件
│   ├── routes/            # 路由配置
│   ├── README/            # 项目文档
│   ├── i18n.ts            # 国际化配置
│   └── main.tsx           # 入口文件
├── package.json
└── README.md
```

## 核心目录说明

### `src/components/` - UI 组件

#### `components/data-table/`
数据表格相关组件，包括：
- `bulk-actions.tsx` - 批量操作
- `column-header.tsx` - 列头
- `faceted-filter.tsx` - 面积过滤器
- `pagination.tsx` - 分页
- `toolbar.tsx` - 工具栏
- `view-options.tsx` - 视图选项

#### `components/layout/`
布局相关组件：
- `app-sidebar.tsx` - 侧边栏
- `app-title.tsx` - 应用标题
- `authenticated-layout.tsx` - 认证布局
- `header.tsx` - 头部
- `main.tsx` - 主内容区
- `nav-group.tsx` - 导航组
- `nav-user.tsx` - 用户导航
- `team-switcher.tsx` - 团队切换器
- `top-nav.tsx` - 顶部导航

### `src/features/` - 功能模块

#### `features/auth/`
认证相关功能：
- `sign-in/` - 登录
- `sign-up/` - 注册
- `forgot-password/` - 忘记密码
- `otp/` - 验证码

#### `features/dashboard/`
仪表盘：
- `components/` - 仪表盘组件
  - `analytics-chart.tsx` - 分析图表
  - `overview.tsx` - 概览
  - `recent-sales.tsx` - 最近销售
- `index.tsx` - 仪表盘页面

#### `features/users/`
用户管理：
- `components/` - 用户组件
  - `users-columns.tsx` - 用户列定义
  - `users-table.tsx` - 用户表格
  - `users-dialogs.tsx` - 用户对话框
- `data/` - 用户数据
  - `users.ts` - 用户数据
  - `schema.ts` - 数据模式
- `index.tsx` - 用户管理页面

#### `features/tasks/`
任务管理：
- `components/` - 任务组件
  - `tasks-columns.tsx` - 任务列定义
  - `tasks-table.tsx` - 任务表格
  - `tasks-dialogs.tsx` - 任务对话框
- `data/` - 任务数据
  - `tasks.ts` - 任务数据
  - `schema.ts` - 数据模式
- `index.tsx` - 任务管理页面

#### `features/settings/`
设置模块：
- `account/` - 账户设置
- `appearance/` - 外观设置
- `display/` - 显示设置
- `notifications/` - 通知设置
- `profile/` - 个人资料
- `index.tsx` - 设置页面

#### `features/chats/`
聊天模块：
- `components/` - 聊天组件
- `data/` - 聊天数据
- `index.tsx` - 聊天页面

#### `features/apps/`
应用管理：
- `data/` - 应用数据
- `index.tsx` - 应用页面

### `src/context/` - 状态管理

- `direction-provider.tsx` - 方向提供者（RTL 支持）
- `font-provider.tsx` - 字体提供者
- `i18n-provider.tsx` - 国际化提供者
- `layout-provider.tsx` - 布局提供者
- `search-provider.tsx` - 搜索提供者
- `theme-provider.tsx` - 主题提供者

### `src/lib/` - 工具库

- `cookies.ts` - Cookie 操作
- `handle-server-error.ts` - 服务器错误处理
- `utils.ts` - 工具函数

### `src/hooks/` - 自定义 Hook

- `use-dialog-state.tsx` - 对话框状态
- `use-mobile.tsx` - 移动端检测
- `use-table-url-state.ts` - 表格 URL 状态

## 路由结构

```
routes/
├── (auth)/
│   ├── forgot-password.tsx
│   └── otp.tsx
└── *.tsx                  # 其他路由
```

## 配置文件

- `src/i18n.ts` - 国际化配置
- `src/main.tsx` - 应用入口
- `package.json` - 项目依赖
- `vite.config.ts` - Vite 配置

## 组件库

- **UI 组件**：Shadcn UI
- **图标**：Lucide Icons
- **构建工具**：Vite
- **路由**：TanStack Router
- **状态管理**：React Context
- **样式**：Tailwind CSS
