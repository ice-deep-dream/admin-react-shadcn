## 技术栈

- **UI 框架**: Shadcn UI + Tailwind CSS
- **构建工具**: Vite
- **路由**: TanStack Router
- **状态管理**: React Context
- **国际化**: i18next + react-i18next
- **类型检查**: TypeScript
- **图标**: Lucide Icons
